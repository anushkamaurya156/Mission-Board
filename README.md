# Mission Board

Mission Board is a lightweight Progressive Web App (PWA) designed to help users organize, track, and complete personal missions and tasks through a clean, distraction-free interface.

## Features

* Create and manage missions
* Track progress visually
* Mark missions as completed
* Responsive design for desktop and mobile
* Installable as a Progressive Web App (PWA)
* Offline support through Service Workers
* Fast and lightweight with no backend required

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Progressive Web App (PWA)
* Service Workers
* Web App Manifest

## Installation

### Clone the repository

```bash
git clone https://github.com/anushkamaurya156/Mission-Board.git
```

### Open the project

```bash
cd Mission-Board
```

### Run locally

Open `index.html` using a local development server such as:

* VS Code Live Server
* Python HTTP Server
* Any static web server

## PWA Support

Mission Board can be installed on:

* Android
* Windows
* macOS
* Linux
* Supported Chromium-based browsers

The application includes:

* Web App Manifest
* Custom icons
* Service Worker caching
* Offline functionality
